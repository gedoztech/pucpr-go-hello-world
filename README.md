# Requirements

- Git: https://git-scm.com/book/en/v2/Getting-Started-Installing-Git

# Instalation

- Clone the repository
```sh
git clone https://github.com/johnny00joe/pucpr-go-hello-world.git
```


# Execution

- Access the project root folder
```sh
cd pucpr-go-hello
```

- Run de program
```sh
go run main.go
```
